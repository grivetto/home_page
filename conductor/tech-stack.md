# Technology Stack

## Frontend
- **Framework:** React (v19.2.0)
- **Build Tool:** Vite (configured with rolldown-vite)
- **State Management:** React Hooks
- **Styling:** CSS3 (Standard CSS files)

## Tooling and Automation
- **Scripting:** Python 3 (for deployment, FTP management, and cleanup)
- **Linting:** ESLint (v9.39.1)
- **Package Manager:** npm

## Deployment
- **Method:** Custom Python-based FTP deployment scripts (`deploy_ftp.py`, `upload_index.py`, etc.)
